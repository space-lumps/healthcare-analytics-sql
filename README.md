# Healthcare Analytics SQL

## Objective
Build a reproducible SQL analysis workflow for a healthcare encounter dataset (cohort definition + derived metrics + QA checks).

## Deliverables
- Cohort definition SQL
- Feature engineering SQL (patient/encounter-level fields)
- QA checks (row counts, duplicates, null rates)
- Example outputs saved to /output (not tracked)

## Repo structure
- /sql     SQL scripts (numbered, runnable in order)
- /docs    Notes, assumptions, data dictionary, validation notes
- /output  Local outputs (ignored)

## How to run
Document the exact tool + commands used (DuckDB, etc.)

## Data
Describe source and schema. Raw data files are not committed.
